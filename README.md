# Generalized-Linear-Models-in-the-Analysis-of-Functional-Near-Infrared-Spectroscopy
W poniższych plikach należy zmienić źródło danych. W celu uzyskania dostępu do danych proszę o kontakt w prywatnej wiadomości (kwestia prywatności danych firmy zewnętrznej).


Pliki .ipynb zawierają następujące kody:
1. Modele liniowe porownanie.ipynb - kod zawierajacy konstrukcje 3 modeli opartych o GLM a następnie przeprowadzenie testu Friedmana w celu porównania wyników.
2. Porownania 6 metod+posthoc.ipynb - kod zawierajacy konstrukcje 3 dodatkowych modeli a nastepne porownanie ich z 3 metodami opartymi o GLM. Porównanie za pomocą testu Friedmana a porównanie parami wykonywane jest za pomocą testu Nemenyi.
3. Analiza odchylen standardowych.ipynb - kod ukazujacy zalety metody AR-IRLS w uwzglednianiu bledu standardowego.
